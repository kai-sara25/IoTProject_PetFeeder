🐾 IoT Pet Feeder Project

An end-to-end IoT solution that allows you to schedule and dispense pet food remotely via a web interface.

Features
Web Dashboard:Simple HTML interface to trigger feeding or view status.
Remote Scheduling:Automated meal times managed by a Flask backend.
Hardware Control:Arduino-based motor/servo control to dispense food.

Tech Stack
Backend:Python 3 with Flask
Frontend: HTML5, CSS
Hardware: Raspberry pi, Raspberry pi touch Display, Arduino (C++), Servo/Motor

📂 Project Structure
/
├── app.py              # Flask server logic
├── templates/
│   └── index.html      # Web dashboard UI
└── ArduinoCode/
    └── PetFeeder.ino   # Firmware for the Arduino board
